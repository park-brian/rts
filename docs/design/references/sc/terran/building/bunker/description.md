# Bunker

Top-down primitive target: a squat octagonal pillbox with a central hatch and four firing slits. The outer silhouette must be defensive and compact.

SVG parts in the current draft:

- `pillbox`: octagonal armor footprint.
- `hatch`: central access circle.
- `north-slit`, `south-slit`, `west-slit`, `east-slit`: four directional firing slits.

Animation/implementation notes: slits can flash from the side that fires, or all slits can pulse for generic occupied-state feedback. The hatch can remain a static identity mark.

Do not draw: interior infantry, wall height, sandbags, shadows, or angled viewport detail.
