# Covert Ops

Top-down primitive target: a low stealth add-on body with two parallel roof bars and a narrow forward visor. It should feel compact, secretive, and rectangular.

SVG parts in the current draft:

- `stealth-body`: main add-on body.
- `left-roof-bar`, `right-roof-bar`: paired vertical bars for cloak/stealth identity.
- `forward-visor`: thin horizontal sensor slit.

Animation/implementation notes: flicker the roof bars for cloak tech and pulse the visor for covert activity. Keep it lower and flatter than the Physics Lab.

Do not draw: spy silhouettes, antennas, height, shadows, or detailed hatches.
