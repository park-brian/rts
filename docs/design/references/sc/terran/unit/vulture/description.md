# Vulture

Reference image: `liquipedia-vulture.jpg`

The reference is a long hover bike, but the top-down glyph should be almost dragster-simple: two narrow forward triangles split down the middle, joined farther back by a longer rectangular body/cabin, with two rear thrusters tucked close to that body. It should feel fast because of length and taper, not because of visible rider detail.

Distinctive features:

- Two long narrow forward triangles, split down the middle.
- Longer rectangular body/cabin set farther back.
- Two slightly wider rectangular rear thrusters, visually above and close to the body.
- Minimal side fins, if any.

SVG plan:

- `panels`: left and right forward triangle, longer rear body/cabin rectangle, then two slightly wider rear thruster rectangles tucked above it.
- `lines`: center split only if the two triangles do not read separately.
- `cores`: none unless a tiny cabin light is needed.
- Make it longer than infantry and narrower than tanks.

Omit:

- Rider anatomy, wheel detail, diamond-shaped body, car body bulk, exhaust flames, and underside shadows.
