# Nuclear Missile

Reference image: `liquipedia-nuclear-missile.gif`

The reference is an icon-like mushroom cloud, but the actual gameplay object is the armed nuke/missile concept. Since it is special and rare, readability can lean symbolic.

Distinctive features:

- Long missile body or warning icon.
- Fins.
- Bright danger core.
- Optional mushroom-cloud command icon if used outside world space.

SVG plan:

- `panels`: long capsule/triangle missile with small fins.
- `lines`: center stripe and fin strokes.
- `cores`: warning glow at nose or center.
- For a command-card icon, a mushroom cloud is acceptable; for world sprite, prefer missile.

Omit:

- Text, radiation symbols, detailed explosion cloud in unit sprite.
