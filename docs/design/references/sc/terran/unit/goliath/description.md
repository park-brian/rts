# Goliath

Reference image: `liquipedia-goliath.jpg`

The reference reads as a bipedal mech, but the SVG sprite should collapse toward the infantry rig in machine form: a bulky square torso/head, square side blocks, and long twin cannons. It is wider and heavier than a Marine, with mechanical squared-off shapes instead of round armor.

Distinctive features:

- Bulky square torso/cockpit mass.
- Two square side blocks, larger than Marine side circles.
- Long twin cannons projecting forward.
- Small square leg/foot hints only if needed for the walker read.

SVG plan:

- `panels`: square torso/head block, two square side blocks, optional tiny foot blocks.
- `lines`: two long parallel cannon strokes; central seam only if readable.
- `cores`: small cockpit/core on the torso.
- Keep it distinct from Marine by making every major shape square and heavier.

Omit:

- Arms, hands, hip joints, tiny missiles, round human circles, and mechanical greebles.
