# Dropship

Reference image: `liquipedia-dropship.jpg`

The reference is a bulky transport with a central body and engines, but the top-down glyph should be simpler: a flying box with a beveled front and integrated thrusters that jut backward. It should read as a transport crate with engines, not a fighter.

Distinctive features:

- Broad rectangular cargo box.
- Beveled front face.
- Integrated left/right thrusters jutting backward from the body.
- Eight center cargo indicator dots in two columns that light up as storage is used.
- Small tail with two side fins, optional if still readable.

SVG plan:

- `panels`: wide central box, beveled nose polygon, two backward thruster blocks, optional tail fin triangles.
- `lines`: one rear ramp seam and simple engine ring/slot strokes.
- `cores`: eight center cargo dots in two columns that light up as the Dropship fills.
- Make it boxier and heavier than Wraith or Valkyrie.

Omit:

- Cargo doors beyond one seam, detached side pods, tiny thrusters, landing gear, and aircraft wing detail.
