# Firebat

Reference image: `liquipedia-firebat.jpg`

The reference is a red heavy infantry suit, but the pure top-down glyph should be a Marine variant: main head circle, bigger side circles, and twin flame bars. Do not draw a separate full character.

Distinctive features:

- Marine-like head circle.
- Bigger side circles than Marine.
- Two wide forward flamethrower bars/nozzles, one attached to each side circle and drawn below the side circles in visual z-order.
- Compact, aggressive stance.

SVG plan:

- `panels`: two wide nozzle rectangles first, then two oversized side circles, then head circle.
- `lines`: two parallel short flamer bars/nozzles attached to those arm blocks.
- `cores`: chest or twin nozzle glow.
- Make the silhouette wider than Marine, not taller.

Omit:

- Full body, legs, boots, backpack detail, loose floating weapons, underside detail, rendered flame effects, and mechanical decoration.
