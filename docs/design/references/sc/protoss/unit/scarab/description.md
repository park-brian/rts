# Scarab

Reference image: none. Liquipedia has no dedicated unit-art image for this projectile.

Treat Scarab as a simple glowing projectile. It does not need creature detail. The player should understand it as a Reaver-launched orb/pellet.

Distinctive features:

- Tiny bright orb.
- Optional small outer ring.
- Very small footprint.
- Projectile identity, not a full unit silhouette.

SVG plan:

- `panels`: none or a tiny dark backing circle.
- `lines`: one small ring.
- `cores`: bright filled circle.
- Keep it smaller than Interceptor and visually close to a particle.

Omit:

- Legs, beetle body, face, and complex shell.
