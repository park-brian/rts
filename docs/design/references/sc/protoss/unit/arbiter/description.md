# Arbiter

Reference image: `liquipedia-arbiter.jpg`

The reference is a crescent/saucer ship with a large side shell and glowing core. It should read as a round caster aircraft.

Distinctive features:

- Crescent or hook-shaped saucer body.
- Central green/psi core.
- One larger shell side.
- Cloak/stasis caster feel.

SVG plan:

- `panels`: crescent hull, small inner body.
- `lines`: circular aura/stasis ring, shell arc.
- `cores`: central caster core.
- Keep it more circular than Corsair/Scout.

Omit:

- Tiny red spikes, detailed cockpit, and many shell panels.
