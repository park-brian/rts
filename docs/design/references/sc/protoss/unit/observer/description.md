# Observer

Reference image: `liquipedia-observer.jpg`

The reference is a small floating eye/drone with side fins. It should read as a tiny detector lens.

Distinctive features:

- Small oval body.
- Central eye/lens.
- Two side fins.
- Cloaked/detector feel.

SVG plan:

- `panels`: oval body with two small fins.
- `lines`: lens ring and one center seam.
- `cores`: central eye core.
- Keep it tiny, clean, and non-combat.

Omit:

- Multiple antennae, underside detail, and weapon cues.
