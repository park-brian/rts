# Reaver

Reference image: `liquipedia-reaver.jpg`

The reference is a bulky beetle-like robotic siege unit with a domed body and many small legs. Simplify to heavy beetle shell and front scarab mouth.

Distinctive features:

- Large oval/domed body.
- Segmented shell plates.
- Small front mouth/launcher.
- Stubby legs around the base.

SVG plan:

- `panels`: large oval body, front launcher plate.
- `lines`: shell segment arcs and a few leg strokes.
- `cores`: central or front launcher glow.
- Keep it slow, wide, and heavy.

Omit:

- Many individual legs, ornate plating, and perspective height.
