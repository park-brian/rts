# Zealot

Reference image: `liquipedia-zealot.jpg`

The reference is a humanoid warrior, but the pure top-down glyph should share the Marine infantry grammar: helmet/shoulders plus weapons. Zealot is a Marine-like body with two sword arcs instead of a gun.

Distinctive features:

- Helmet/head at upper center.
- Two shoulder/pauldron masses.
- Two forward psi-sword arcs, one on each side.
- Tiny torso connector only if needed.

SVG plan:

- `panels`: helmet/torso nub, two shoulder plates.
- `lines`: two clean forward sword arcs, mirrored left/right.
- `cores`: chest/psi core.
- Make it taller than Marine but still simple.

Omit:

- Legs, feet, face, hands, cloth, underside detail, full body anatomy, and blade texture.
