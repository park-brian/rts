# Dragoon

Reference image: `liquipedia-dragoon.jpg`

The reference is a four-legged walker with a rounded gold shell and red core. The pure top-down identity is simpler: orb/shell plus four legs.

Distinctive features:

- Central orb/shell.
- Four splayed legs.
- Bright center core.
- Walker, not tank.

SVG plan:

- `panels`: round/oval central orb shell.
- `lines`: four leg strokes/plates, one shell arc at most.
- `cores`: center core.
- Keep the shell broad and insect-mech like.

Omit:

- Extra leg joints, face details, underside detail, and complex gold plating.
