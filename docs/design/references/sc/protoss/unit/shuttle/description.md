# Shuttle

Reference image: `liquipedia-shuttle.jpg`

The reference is a curved transport shell with a crescent body and open underside. It should read as a Protoss transport: rounded, hollow, and wider than a fighter.

Distinctive features:

- Crescent/oval transport hull.
- Open central cargo bay.
- Two side nacelle-like curves.
- Smooth gold shell language.

SVG plan:

- `panels`: crescent or horseshoe hull.
- `lines`: inner cargo bay arc, side nacelle seams.
- `cores`: small forward core.
- Make it rounder and less aggressive than Scout/Corsair.

Omit:

- Interior detail, passengers, and many small shell plates.
