# Dark Archon

Reference image: `liquipedia-dark-archon.jpg`

The reference is a dark energy mass. For top-down SVG, make this a glowing ball like Archon, but with a black core. The surrounding glow is always the team color.

Distinctive features:

- Large glowing central sphere.
- Black core, always black regardless of team color.
- Team-color outer glow/aura.
- Caster, no weapon, no visible body.

SVG plan:

- `panels`: small black center disk/orb.
- `lines`: one or two circular aura rings only if needed.
- `cores`: large team-color orb/glow around the black center.
- Differentiate from Archon by core color: Archon has blue core, Dark Archon has black core.

Omit:

- Dense fog, many particles, humanoid details, arms, shoulder shapes, and shard clutter.
