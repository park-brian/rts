# Archon

Reference image: `liquipedia-archon.jpg`

The reference is mostly energy. For top-down SVG, make this a glowing ball, not a humanoid. The core identity is a bright blue center with a larger team-color glow around it.

Distinctive features:

- Large glowing central sphere.
- Blue core, always blue regardless of team color.
- Team-color outer glow/aura.
- Bigger than normal infantry and visually soft-edged.

SVG plan:

- `panels`: none unless a tiny inner blue panel helps the core read.
- `lines`: one or two circular aura rings only if needed.
- `cores`: very large team-color orb with a smaller blue center.
- Let the ball shape dominate; this should read as a glowing sphere at game scale.

Omit:

- Humanoid anatomy, arms, shoulders, shard clutter, face/body details, and many lightning strokes.
