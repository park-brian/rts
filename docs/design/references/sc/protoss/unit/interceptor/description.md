# Interceptor

Reference image: `liquipedia-interceptor.jpg`

The reference is a tiny red/gold drone fighter. It should be a small diamond dart.

Distinctive features:

- Tiny wedge/diamond body.
- Two small side fins.
- Single core/cockpit.
- Fast projectile-like fighter.

SVG plan:

- `panels`: small diamond or dart polygon.
- `lines`: two fin strokes.
- `cores`: tiny center core.
- Keep it larger than Scarab but smaller than Probe.

Omit:

- Detailed wings, cockpit frame, and multiple panels.
