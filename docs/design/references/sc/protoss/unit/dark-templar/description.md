# Dark Templar

Reference image: `liquipedia-dark-templar.jpg`

The reference is a dark cloaked melee unit. In SVG, use an angular cloak wedge plus one blade crescent.

Distinctive features:

- Narrow cloaked body.
- Angular hood/head.
- One strong warp blade crescent.
- Stealthier and sharper than Zealot.

SVG plan:

- `panels`: tall dark diamond/cloak wedge.
- `lines`: single long curved blade, small hood/eye line.
- `cores`: tiny hidden psi core.
- Keep silhouette asymmetric enough to separate from Zealot.

Omit:

- Face, cloth folds, two full arms, and detailed armor.
