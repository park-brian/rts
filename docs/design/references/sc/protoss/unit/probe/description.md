# Probe

Reference image: `liquipedia-probe.jpg`

The reference is a tiny floating worker, but the isometric angle distorts it. Read through the perspective: the sprite should just be a forward-pointing V. Keep the point facing forward/up in our SVG source.

Distinctive features:

- Forward-pointing V body.
- Small core dot near the inside of the V only if needed.
- Floating, no legs.

SVG plan:

- `panels`: one shallow V-shaped shell, not a diamond and not a four-point star.
- `lines`: none unless a center seam is needed.
- `cores`: optional tiny dot tucked inside the V.
- Keep it delicate and smaller than combat units.

Omit:

- Tiny appendages, worker beam, perspective underside, red cap detail, rear prongs, and generic four-point star symmetry.
