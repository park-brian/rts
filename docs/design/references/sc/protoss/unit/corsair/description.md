# Corsair

Reference image: `liquipedia-corsair.jpg`

The reference is a small crescent fighter. It should be thin, curved, and fast, with split wing tips.

Distinctive features:

- Curved crescent body.
- Split forward/side tips.
- Small blue core/engine.
- Compact air-to-air identity.

SVG plan:

- `panels`: thin crescent/hook hull.
- `lines`: split-tip strokes and center seam.
- `cores`: small engine/core.
- Keep it narrower and lighter than Scout.

Omit:

- Many wing plates, cockpit detail, and background energy effects.
