# Carrier

Reference image: `liquipedia-carrier.jpg`

The reference is a huge curved capital ship with a long oval hull and side bays. The sprite must feel large and smooth.

Distinctive features:

- Large elongated oval/capsule hull.
- Side bay arcs for Interceptors.
- Front bridge/core.
- Smooth capital ship mass.

SVG plan:

- `panels`: large oval/capsule hull, side bay shapes.
- `lines`: bay arcs, center spine, front core ring.
- `cores`: large bridge/core.
- Make it the largest Protoss air silhouette.

Omit:

- Individual Interceptors inside the hull, tiny gold plates, and perspective height.
