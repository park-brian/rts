# Zergling

Reference image: `liquipedia-zergling.jpg`

The reference is a tiny fast melee creature. The pure top-down glyph should be reduced to jaws, a slightly longer torso, and two forward claw arcs.

Distinctive features:

- Open jaw shape at the front.
- Longer narrow torso behind the jaws.
- Two forward claw arcs.
- Low, fast posture.

SVG plan:

- `panels`: narrow torso teardrop and simple jaw/head wedge.
- `lines`: two forward claw arcs. Optional tiny jaw line.
- `cores`: tiny eye/core.
- Keep it smaller and sharper than Broodling.

Omit:

- Many legs, feet, teeth rows, horns, underside detail, and detailed carapace.
