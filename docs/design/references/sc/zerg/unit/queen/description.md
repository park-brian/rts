# Queen

Reference image: `liquipedia-queen.jpg`

The reference is a floating caster with a long tail, side wings, and claw forms. It should read as insect queen/caster, not combat flyer.

Distinctive features:

- Central insect body.
- Long tail/abdomen.
- Side wings or claw fins.
- Head/core at front.

SVG plan:

- `panels`: oval thorax and long tail teardrop.
- `lines`: side wing/claw curves, tail seam.
- `cores`: head/caster core.
- Keep it elegant and elongated.
