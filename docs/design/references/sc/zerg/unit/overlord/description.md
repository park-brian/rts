# Overlord

Reference image: `liquipedia-overlord.jpg`

The reference is a large floating sac with horns/tentacles. It should be the roundest Zerg air unit and obviously non-weaponized.

Distinctive features:

- Large bloated central sac.
- Horns or tusk-like side forms.
- Hanging tentacles.
- Eye/core on the front.

SVG plan:

- `panels`: large oval body, horn plates.
- `lines`: tentacle strokes, body seam arcs.
- `cores`: central eye/core.
- Make it large and slow-looking.

Omit:

- Many tentacles, surface bumps, and detailed mouth parts.
