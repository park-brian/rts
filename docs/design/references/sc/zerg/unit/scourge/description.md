# Scourge

Reference image: `liquipedia-scourge.jpg`

The reference is a tiny suicide flyer. It should be a small winged dart, more fragile than Mutalisk.

Distinctive features:

- Tiny central body.
- Two small wings.
- Split tail or stinger.
- Projectile-like profile.

SVG plan:

- `panels`: small body wedge and wing pair.
- `lines`: tail split, one wing rib per side.
- `cores`: tiny center core.
- Keep it about Interceptor scale.

Omit:

- Detailed head, claws, and multiple body segments.
