# Defiler

Reference image: `liquipedia-defiler.jpg`

The reference is a low caster with a long curved body/tail and spines. In our existing style, it can borrow heavily from the current Sunken Colony defense sprite language, essentially flipped around: spined low body, organic root/tail mass, but with a caster core instead of a static building base.

Distinctive features:

- Low spined body.
- Sunken Colony-like spine/tail language, flipped in facing.
- Small caster head/core.
- Tentacle/leg hints only if they help the silhouette.

SVG plan:

- `panels`: low tapered body and head plate, borrowing Sunken Colony proportions.
- `lines`: curved spine and two tentacle/root strokes; flip the Sunken Colony read so it faces as a mobile unit.
- `cores`: small head/caster core.
- Keep it lower and stranger than Hydralisk.

Omit:

- Many legs, texture, static building footprint, and full underside.
