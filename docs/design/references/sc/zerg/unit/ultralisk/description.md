# Ultralisk

Reference image: `liquipedia-ultralisk.jpg`

The reference is a massive horned melee beast with huge curved tusks. The tusks are the unit.

Distinctive features:

- Huge central body.
- Two giant curved tusks/scythes.
- Broad head/front.
- Heavy rear mass.

SVG plan:

- `panels`: large body oval, head plate.
- `lines`: two oversized tusk arcs, carapace seam.
- `cores`: head/chest core.
- Make it the largest ground unit silhouette.

Omit:

- Small legs, teeth, surface texture, and many shell plates.
