# Drone

Reference image: `liquipedia-drone.jpg`

The reference is a small worker insect, but the top-down sprite should be much simpler: an oval body with two side wings. Do not over-describe claws or legs; those are projection/detail traps at gameplay scale.

Distinctive features:

- Oval body.
- Two side wings.
- Small forward point/core only if needed.
- Round, non-aggressive worker read.

SVG plan:

- `panels`: central oval body plus two simple wing ovals/arcs.
- `lines`: one short center seam only if it improves orientation.
- `cores`: optional small front/core dot.
- Keep it compact and round.

Omit:

- Legs, mandibles, harvesting claws, fine wing veins, surface texture, shadow, and underside.
