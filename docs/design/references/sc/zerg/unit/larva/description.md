# Larva

Reference image: `liquipedia-larva.jpg`

The reference is a tiny red grub. It should be very small and curled, but still visible on creep.

Distinctive features:

- Curled grub body.
- Segmented back.
- Tiny head dot.
- No legs.

SVG plan:

- `panels`: small curved bean/teardrop body.
- `lines`: two or three segment arcs.
- `cores`: tiny head/core dot.
- Keep it smaller than Drone and Zergling.

Omit:

- Detailed mandibles, many body rings, and shadows.
