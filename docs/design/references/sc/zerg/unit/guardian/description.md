# Guardian

Reference image: `liquipedia-guardian.jpg`

The reference is a large flying crab-like creature with big claws/wings and a central body. It should feel heavier than Mutalisk.

Distinctive features:

- Wide crab-like wing/claw arms.
- Heavy central abdomen.
- Front head/core.
- Slow siege flyer posture.

SVG plan:

- `panels`: central body plus two large curved side claws/wings.
- `lines`: carapace segments and claw arcs.
- `cores`: front/head core.
- Make it broad and heavy.

Omit:

- Small legs, detailed shell ridges, and texture.
