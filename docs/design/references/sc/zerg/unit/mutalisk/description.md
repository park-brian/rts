# Mutalisk

Reference image: `liquipedia-mutalisk.jpg`

The reference is a bat-like flying creature with two large red wings and a narrow body. This should be the cleanest Zerg air silhouette.

Distinctive features:

- Two large bat wings.
- Narrow central body.
- Tail or stinger.
- Small head.

SVG plan:

- `panels`: left/right wing membranes, central body.
- `lines`: wing rib lines, tail stroke.
- `cores`: small head/chest core.
- Keep it wide and lightweight.

Omit:

- Vein texture, teeth, many wing ribs, and perspective folds.
