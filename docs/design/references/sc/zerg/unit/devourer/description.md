# Devourer

Reference image: `liquipedia-devourer.jpg`

The reference is a bulky air beast with a massive shell and purple underside. It should be the largest, fattest Zerg air attacker.

Distinctive features:

- Bulky armored carapace.
- Large forward maw/horn.
- Side wing fins.
- Heavy abdomen.

SVG plan:

- `panels`: large oval shell, front maw plate, side fins.
- `lines`: shell segment arcs, maw line.
- `cores`: small acidic mouth/core.
- Make it chunkier than Guardian and Mutalisk.

Omit:

- Purple underside ribs beyond one or two segment lines.
