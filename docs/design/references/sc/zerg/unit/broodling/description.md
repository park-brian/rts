# Broodling

Reference image: `liquipedia-broodling.jpg`

The reference is a tiny spawned melee creature. It should be simpler and smaller than Zergling.

Distinctive features:

- Tiny body.
- Small head.
- Two claw limbs.
- Short-lived spawned-unit feel.

SVG plan:

- `panels`: small teardrop body and head.
- `lines`: two small claw arcs.
- `cores`: tiny head/core.
- Keep it very small but readable.

Omit:

- Full Zergling scythes, many legs, and detailed carapace.
