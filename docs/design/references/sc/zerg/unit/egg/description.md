# Egg

Reference image: `liquipedia-egg.jpg`

The reference is a squat cocoon with several glowing sacs. It should read as an immobile morphing object.

Distinctive features:

- Oval cocoon mass.
- Several rounded carapace/sac plates.
- Organic seam lines.
- Stationary footprint.

SVG plan:

- `panels`: large oval/egg body, two or three sac plates.
- `lines`: vein/seam arcs.
- `cores`: one inner glow/slit.
- Keep it round and inert.

Omit:

- Many small bumps, slime texture, and hatch animation detail.
